## Python 가상 환경 만드는 법

### Create virtualenv
```console
$ conda create -n your_envname
```

### Activate virtualenv
```console
$ conda activate your_envname
```

### Deactivate virtualenv
```console
$ conda deactivate
```

### Remove virtualenv
```console
$ conda env remove -n your_envname
```
